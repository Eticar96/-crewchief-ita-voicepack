# TESTO DI REGISTRAZIONE VOCE — LEONARDO

Copione completo per la registrazione della voce "Leonardo", ingegnere di pista italiano per CrewChief V4.
Durata totale stimata: **8-10 minuti** di audio pulito.

---

## ISTRUZIONI GENERALI PER LA REGISTRAZIONE

### Formato e file
- **Registra in un unico file lungo per blocco** (4 file totali: leonardo_A01.wav, leonardo_B01.wav, leonardo_C01.wav, leonardo_D01.wav)
- Formato: **WAV, 44100 Hz, 16 bit, mono**
- Se il tuo DAW registra in stereo, convertilo in mono dopo
- Nomina i file esattamente come indicato (leonardo_A01.wav ecc.)

### Microfono e ambiente
- Distanza dal microfono: **15-20 cm**, mantienila costante per tutta la sessione
- Usa un **filtro anti-pop** se disponibile
- Stanza silenziosa, niente ventilatori, finestre chiuse
- Niente vestiti che frusciano, niente bracciali, orologi che toccano il tavolo

### Pause e respiri
- Dove vedi **[PAUSA 3 SEC]**: fermati 3 secondi in silenzio, poi riprendi
- Dove vedi **[RESPIRO]**: prendi fiato normalmente, non trattenerlo
- I respiri naturali tra le frasi **vanno benissimo** — NON cercare di eliminarli, il software li gestisce
- Tra un blocco e l'altro: **fermati, bevi acqua, riposati 2-3 minuti**

### Se sbagli una frase
- **NON fermarti e ricominciare da capo**
- Fai una pausa di 3 secondi, poi **ripeti SOLO la sezione sbagliata** dall'inizio
- Il silenzio di 3 secondi ci permette di tagliare facilmente in post-produzione

### Ritmo e naturalezza
- Pensa a una **vera comunicazione radio** durante una gara: concisa ma chiara
- NON correre: le parole devono essere scandite
- NON trascinare: non allungare le vocali come se stessi leggendo una poesia
- Il ritmo giusto e' quello di una **conversazione tecnica tra professionisti**
- Immagina di parlare in cuffia a un pilota che sta guidando a 300 km/h: deve capire al primo colpo

### Bere acqua
- **Bevi un sorso d'acqua** tra un blocco e l'altro (ogni 15-20 frasi)
- Se senti la voce che si secca, fermati e bevi anche a meta' blocco
- Voce idratata = voce migliore
