# REGISTRAZIONE VOCE V2 — LEONARDO

Copione per migliorare il clone vocale. File CORTI (6-10 secondi),
piu' precisi per XTTS v2. Si aggiungono ai 13 reference esistenti.

---

## REGOLE (leggile TUTTE prima di registrare)

1. **Distanza microfono: 10-12 cm** — piu' vicino di prima
2. **Volume ALTO e COSTANTE** — come parlare in un box rumoroso
3. **ZERO pause** — finisci una frase e attacca subito la successiva
4. **Voce dal petto** — mento leggermente abbassato
5. **Ogni file: 6-10 secondi** di parlato continuo, non di piu'
6. **Formato: WAV, 44100 Hz, 16-bit, mono**
7. **Se sbagli:** non fermarti, rifai il file da capo
8. **Bevi acqua** tra un blocco e l'altro

---
