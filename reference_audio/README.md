# Audio di Riferimento per Clonazione Vocale

Questa cartella contiene gli audio di riferimento per la clonazione vocale con XTTS v2.

## Struttura

```
reference_audio/
├── marco/      # Audio per voce "Marco"
├── gianni/     # Audio per voce "Gianni"
└── chiara/     # Audio per voce "Chiara"
```

## Requisiti audio

- **Durata**: 20-30 secondi totali (uno o più file)
- **Formato**: WAV, MP3 o FLAC
- **Qualità**: registrazione pulita, senza rumore di fondo
- **Contenuto**: parlato naturale in italiano, possibilmente con tono da ingegnere/tecnico

Vedi `docs/RECORDING_GUIDE.md` per istruzioni dettagliate sulla registrazione.
