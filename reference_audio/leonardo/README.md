# Reference Audio — Leonardo

Metti qui i file audio registrati seguendo il copione in `docs/TESTO_REGISTRAZIONE_VOCE.md`.

## File richiesti

| File | Blocco | Contenuto |
|------|--------|-----------|
| `leonardo_A01.wav` | Tecnico | 15 sezioni, voce ferma e professionale |
| `leonardo_B01.wav` | Spotter | 10 sezioni, voce secca e rapida |
| `leonardo_C01.wav` | Emotivo | 10 sezioni, voce carica e entusiasta |
| `leonardo_D01.wav` | Arrabbiato | 5 sezioni, voce irritata e frustrata |

## Formato

- WAV, 44100 Hz, 16 bit, mono
- Durata totale: 8-10 minuti
- Niente rumore di fondo
- Distanza microfono: 15-20 cm

## Dopo la registrazione

Lancia `scripts/genera_leonardo.bat` per generare il voice pack.
